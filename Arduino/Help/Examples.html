NNet Library examples

In this page we can see some application examples of library usage. Arduino sketches are included in examples directory.

Training has been made on PC for fast process. After training, Neural Networks have been saved in PROGMEM format and in this way NN are included into the sketches.

  • XOR and basic example
  • Sensors management
  • Control system
  • MNIST test (handwritten digit)(example of using NNlib on PC;not for Arduino Uno)
